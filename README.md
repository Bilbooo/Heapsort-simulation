# ⚡ HeapSort Simulator
Un simulatore interattivo per visualizzare il funzionamento dell'algoritmo HeapSort.

Link al simulatore:
https://bilbooo.github.io/Heapsort-simulation/

## 🛠️ Comandi
- **Spazio / Next**: Avanza di un passo
- **P**: Play / Pausa
- **Frecce**: Vai avanti o indietro velocemente
